# Barbearia Corte Fino

Landing page para uma barbearia ficticia, desenvolvida como projeto do checkpoint da trilha **Primeiros Passos**.

O site apresenta a Barbearia Corte Fino, com atendimento por agendamento, servicos de corte e barba, informacoes de contato e localizacao.

## Destaques do projeto

- Visual profissional com paleta em carvao, marfim e bronze
- Fotografias fotorrealistas da barbearia e do atendimento
- Topo fixo com logomarca, navegacao interna e botao de agendamento
- Area principal em tela cheia com chamada para reserva de horario
- Cartoes para apresentar os servicos oferecidos
- Botao flutuante para voltar ao topo
- Layout responsivo para telas menores

## Secoes do site

1. Topo
2. Area principal
3. Servicos
4. Sobre nos
5. Contato com mapa do Google Maps

## Tecnologias e recursos

- HTML5 com tags semanticas: `header`, `main`, `section`, `nav`, `article`, `figure`, `img`, `iframe` e `span`
- CSS3 para cores, tipografia, grade de conteudo, responsividade e transicoes
- Links internos para navegacao entre as secoes
- Fontes aplicadas por classes em elementos `div`
- Imagens profissionais criadas para compor o banner e a secao Sobre nos

## Como visualizar

Abra o arquivo `index.html` em um navegador.
